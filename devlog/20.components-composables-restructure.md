# 20. 重构 components 和 composables 目录结构

## 目标

将 `components/docs-page/src/` 中的独立组件和 composable 按照项目规范分离到独立目录。

## 执行内容

### 组件目录重构

将独立组件从 `docs-page` 中分离，按照 `create-vue-component` 技能规范重构：

| 原路径 | 新路径 |
| --- | --- |
| `components/docs-page/src/DocsNav.vue` | `components/docs-nav/` |
| `components/docs-page/src/DocsSurround.vue` | `components/docs-surround/` |
| `components/docs-page/src/DocsToc.vue` | `components/docs-toc/` |

每个组件目录结构：
```
components/<component-name>/
├── index.ts           # 入口文件
└── src/
    ├── index.vue      # Vue SFC
    ├── ctx.ts         # Props 和 Emits 定义
    └── types.ts       # TypeScript 类型定义
```

### Composable 目录重构

| 原路径 | 新路径 |
| --- | --- |
| `components/docs-page/src/useScrollspy.ts` | `composables/scrollspy/index.ts` |

### 配置更新

1. **根目录 package.json**：添加 `@vike-vue-content/composables` 工作区依赖
2. **pnpm-workspace.yaml**：添加 `composables` 目录到工作区
3. **composables/package.json**：包名从 `@vunk-template/composables` 改为 `@vike-vue-content/composables`
4. **entry/package.json**：添加新组件和 composable 的 exports 配置
5. **README.md**：更新导出表格

### 导入路径更新

`components/docs-page/src/index.vue` 更新为：
```typescript
import ContentRenderer from '../../content-renderer'
import DocsNav from '../../docs-nav'
import DocsToc from '../../docs-toc'
import DocsSurround from '../../docs-surround'
import { useScrollspy } from '@vike-vue-content/composables/scrollspy'
```
