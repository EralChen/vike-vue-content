# 2026-06-01: 建立 shared/types 公共类型入口

## 当前标准

- `vike-vue-content/shared/types` 不再是空模板，开始承担库的公共类型出口。
- 当前对外暴露的核心类型包括：
  - `ContentEntry`
  - `ContentNavigationItem`
  - `ContentDirectoryConfig`
  - `ContentNavigationConfig`
  - `QueryOptions`
  - `QueryBuilder`
  - `DocsPageOptions`
  - `ResolvedDocsPageOptions`
  - `DocsPageData`

## 关键决策

- 现阶段只把 `shared/types` 做成稳定发布入口，不让 `query`、`config`、`components` 在源码层直接依赖 `shared/`。
- 原因是当前 `pnpm-workspace.yaml` 并没有把 `shared/` 当作正式 workspace 包安装管理；直接引入 `workspace:*` 依赖会让构建在依赖解析阶段失败。
- `ContentEntry.body` 在 `shared/types` 边界上保持为 `unknown`，不把 `comark` AST 类型泄漏成共享包的硬依赖。

## 验证记录

通过：

```bash
pnpm -C entry run build
```

结果：

- `shared/types` 成功生成声明产物并被同步到 `entry/shared/types/`。
- `config`、`query`、`components`、`composables`、`shared` 整体构建通过。
- 发布包 README 同步更新，`shared/types` 的用途不再描述为空模板。
