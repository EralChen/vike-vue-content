# CodePreview

`code-preview` renders a live preview above (or beside) the source code. It uses two slots: the default slot for the preview, and an optional `#code` slot for the source.

Built into `ContentRenderer` — no registration needed.

## Basic usage

````md
::code-group
```vue [Code]
<template>
  <span class="badge">New</span>
</template>
```

:::code-preview
<span class="badge">New</span>

#code
```vue
<template>
  <span class="badge">New</span>
</template>
```
:::
::
````

::code-group
```vue [Code]
<template>
  <span class="badge">New</span>
</template>
```

:::code-preview
<span class="badge" style="display:inline-block;padding:2px 8px;border-radius:9999px;background:#3b82f6;color:white;font-size:12px;">New</span>

#code
```vue
<template>
  <span class="badge">New</span>
</template>
```
:::
::

## Slots

| Slot | Description |
| --- | --- |
| `default` | Live rendered preview content. |
| `code` | (Optional) Source code displayed below the preview. Omit for preview-only. |

## Nesting

Colon count follows the [MDC nesting convention](https://remark-mdc.nuxt.space/#syntax) — when `code-preview` is nested inside `code-group`, it uses one more colon level than its parent.
