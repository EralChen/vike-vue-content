# 内容驱动路由

`content/docs/**/*.md` 会按目录结构映射为页面路由：

| 内容文件 | 访问路径 |
| --- | --- |
| `content/docs/index.md` | `/docs` |
| `content/docs/getting-started.md` | `/docs/getting-started` |
| `content/docs/guide/routing.md` | `/docs/guide/routing` |

本页位于嵌套目录 `guide/`，并使用了 `1.` 前缀来对标 Nuxt Content 的文件排序规则。当前实现里，数字前缀会参与排序，但不会出现在最终路由里。

## 路由展开

页面路径由内容目录自动生成，目录和文件上的数字前缀会在路径阶段被剥离。

## 预渲染枚举

`queryCollectionPaths()` 会返回去掉排序前缀后的最终公开路径，用于 docs prerender。
