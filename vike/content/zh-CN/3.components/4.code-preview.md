# CodePreview

`code-preview` 在源码上方渲染实时预览。使用两个插槽：默认插槽放预览内容，可选的 `#code` 插槽放源码。

已内置于 `ContentRenderer`，无需额外注册。

## 基本用法

````md
::code-group
```vue [Code]
<template>
  <span class="badge">New</span>
</template>
```

:::code-preview
<span class="badge">New</span>

#code
```vue
<template>
  <span class="badge">New</span>
</template>
```
:::
::
````

::code-group
```vue [Code]
<template>
  <span class="badge">New</span>
</template>
```

:::code-preview
<span class="badge" style="display:inline-block;padding:2px 8px;border-radius:9999px;background:#3b82f6;color:white;font-size:12px;">New</span>

#code
```vue
<template>
  <span class="badge">New</span>
</template>
```
:::
::

## 插槽

| 插槽 | 说明 |
| --- | --- |
| `default` | 实时渲染的预览内容。 |
| `code` | （可选）显示在预览下方的源码。省略则仅展示预览。 |

## 嵌套

冒号数量遵循 [MDC 嵌套规范](https://remark-mdc.nuxt.space/#syntax) —— `code-preview` 嵌套在 `code-group` 内时，比父级多一层冒号。
