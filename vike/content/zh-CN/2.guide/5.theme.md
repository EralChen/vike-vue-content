# 主题系统

内置主题系统提供主色调、配色模式、圆角和字体自定义。接入 config 后会自动注入一段内联初始化脚本，在首屏前就应用好主题，避免页面切换或刷新时的闪烁（无 FOUC）。

## 使用主题组件

框架提供两个现成组件：`ThemeToggle`（明暗切换按钮）和 `ThemeSettings`（完整主题面板）。

```vue
<script setup>
import { ThemeToggle } from 'vike-vue-content/components/theme-toggle'
import { ThemeSettings } from 'vike-vue-content/components/theme-settings'
</script>

<template>
  <header>
    <h1>My App</h1>
    <div>
      <ThemeSettings />
      <ThemeToggle />
    </div>
  </header>
</template>
```

> 记得在布局里引入样式：`import 'vike-vue-content/index.css'`。该文件已包含所有 CSS 变量定义，无需重复声明。

## 可定制项

| 维度 | 可选值 |
| --- | --- |
| Primary（主色） | `red` `orange` `amber` `yellow` `lime` `green` `emerald` `teal` `cyan` `sky` `blue` `indigo` `violet` `purple` `fuchsia` `pink` `rose`（17 种）、`#0066cc` 这类 raw hex |
| Neutral（中性色） | `slate` `gray` `zinc` `neutral` `stone`（5 种） |
| Radius（圆角） | `0` `0.125` `0.25` `0.375` `0.5`（5 档预设） |
| Font（字体） | `Inter` `system-ui` `Roboto` `Open Sans` `Montserrat` `Poppins` `Outfit` `Raleway`（8 种） |
| Color Mode（配色模式） | 浅色 / 深色 / 跟随系统 |

`ThemeSettings` 面板还支持导出 CSS 或 JSON 配置，以及一键重置。

## 编程式控制

用 `useTheme()` composable 在代码里读写主题：

```vue
<script setup>
import { useTheme } from 'vike-vue-content/composables/theme'

const {
  theme,           // 响应式主题配置
  isDark,          // 是否深色模式
  primary,         // 主色（可读写）
  neutral,         // 中性色（可读写）
  radius,          // 圆角（可读写）
  font,            // 字体（可读写）
  mode,            // 配色模式：'light' | 'dark' | 'system'
  toggleDarkMode,  // 切换深色模式
  resetTheme,      // 重置主题
  exportCSS,       // 导出 CSS 字符串
  exportConfig,    // 导出 JSON 配置字符串
} = useTheme()
</script>
```

`primary`、`neutral`、`radius`、`font`、`mode` 都是可写的 computed，直接赋值即可生效：

```ts
primary.value = 'violet'
primary.value = '#0066cc'
mode.value = 'dark'
radius.value = 0.5
```

命名主色使用内置的 3 阶色板。Raw hex 支持 `#rgb` 和 `#rrggbb`；主题系统会规范化色值，并使用 Chroma.js 自动生成 `--color-primary-light` / `--color-primary-dark`。

主题配置的结构：

```ts
interface ThemeConfig {
  colors: { primary: string; neutral: string }
  radius: number
  font: string
  colorMode: 'light' | 'dark' | 'system'
}
```

## 使用建议

布局组件中直接引用变量即可，无需为深色模式写重复的覆盖样式：

```vue
<style scoped>
.card {
  background: var(--color-bg);
  color: var(--color-text);
  border: 1px solid var(--color-border);
  border-radius: var(--radius);
}

.card:hover {
  background: var(--color-surface-elevated);
}

/* 半透明背景可以用 color-mix */
.header {
  background: color-mix(in srgb, var(--color-bg) 75%, transparent);
  backdrop-filter: blur(12px);
}
</style>
```

框架会在根节点的 `.dark` 类下自动切换所有变量值，因此上面的写法天然兼容深色模式。
